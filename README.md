## 💻 How to Run Locally

1. Clone the repository  
```bash
git clone https://github.com/neelanshu7/Interactive_Quiz_Application.git
```
2. Navigate to the folder
```bash
cd Interactive_Quiz_Application
```
3. Open index.html in your browser
```bash
start index.html
```
